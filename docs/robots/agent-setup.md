# Agent and Prompt Setup

## Overview
This document describes the AI agent and prompt configuration for the Steam game development project.

## Directory Structure
```
.github/
├── agents/          # Agent configuration files
│   └── @idunno.json
└── prompts/         # Prompt templates
    └── @idunno.md
```

## Agent Configuration
The `@idunno.json` defines the AI agent's capabilities and context for assisting with game development tasks.

### Capabilities
- Game design and mechanics
- Level design
- Asset management
- Performance optimization
- Steam integration
- Playtesting feedback

## Prompt Template
The `@idunno.md` provides guidance for AI-assisted game development, focusing on:
- Gameplay-first approach
- Performance optimization
- Steam platform integration
- Community engagement

## Usage
These configurations help guide AI tools in understanding the project context and providing relevant assistance for Steam game development.
