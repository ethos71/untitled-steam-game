# Shell Level System

## Overview
The world is divided into multiple shells, each representing a layer descending toward the core. Each shell has unique characteristics, enemies, and challenges inspired by Dante's Inferno.

## Shell Structure

### Shell 1: The Mist Surface
- **Theme**: Limbo/Beginning
- **Environment**: Floating spires above the deadly mist
- **Enemies**: Basic corrupted creatures
- **Challenges**: Learning to survive, basic combat

### Shell 2: The Industrial Depths
- **Theme**: Greed/Industry
- **Environment**: Abandoned factories and mining operations
- **Enemies**: Mechanized monsters, corrupted workers
- **Challenges**: Steampunk machinery puzzles

### Shell 3: The Forgotten Cities
- **Theme**: Gluttony/Excess
- **Environment**: Ruins of ancient civilization
- **Enemies**: Mutated citizens, resource hoarders
- **Challenges**: Urban exploration, scavenging

### Shell 4: The Burning Wastes
- **Theme**: Wrath/Violence
- **Environment**: Scorched battlefields and war-torn landscapes
- **Enemies**: War machines, berserkers
- **Challenges**: Environmental hazards, combat gauntlets

### Shell 5: The Frozen Betrayal
- **Theme**: Treachery/Deceit
- **Environment**: Ice caverns and frozen ruins
- **Enemies**: Illusion-based enemies, backstabbers
- **Challenges**: Trust mechanics, deception puzzles

### Shell 6: The Corrupted Core
- **Theme**: The Source/Final Circle
- **Environment**: The heart of the apocalypse
- **Enemies**: Ancient legendary beings
- **Challenges**: Ultimate boss battles, truth revelation

## Implementation Notes
- Each shell is self-contained with its own enemy types, items, and story elements
- Progressive difficulty as you descend
- Each shell reveals more about the pre-apocalypse world
- Boss fights at the end of each shell
